# Model Optimisation / Enhancement Questions

- What are the main pain points with the current model? (too slow, hard to use, errors, inflexible?)
- What works well that should be preserved?
- Is this a cosmetic cleanup or a structural rebuild?
- Can the existing structure be enhanced, or does it need a fresh start?
- Are there circular references? How are they currently handled?
- File size — is it bloated? (hidden data, unused sheets, embedded objects?)
- Who else uses the model? Will changes disrupt their workflows?
- Is there documentation for the current model?
