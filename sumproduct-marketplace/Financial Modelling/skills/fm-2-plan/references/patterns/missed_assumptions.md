# Commonly Missed Assumptions

Use during Stage 3 (section layout): for each item the scope DOES include,
proactively suggest the companion assumption users routinely forget.

| If Scope Includes | Suggest Also Including |
|------------------|----------------------|
| Revenue | Revenue escalation/inflation assumption |
| Debt with floating rate | Interest rate forecast or swap rate |
| CapEx | Contingency % on CapEx |
| Multi-year project | Inflation rate for cost escalation |
| Inventory | Minimum stock days, reorder quantities |
| Tax losses | Loss utilisation cap |
| Construction phase | Liquidated damages risk, delay buffer |
| Foreign revenue | FX rate assumptions, hedging |
| Headcount-driven costs | Annual raises, on-costs %, FTE phasing |
| Commission plans | Clawback provisions, quota changes mid-year |
| Departmental budgets | Allocation keys for shared costs |
| Weekly cashflow | Committed vs discretionary split, facility headroom threshold |
| DCF valuation | Mid-year convention choice, g < WACC validation |
| Scenarios | Which assumptions vary per scenario (completeness per case) |

Frame it as: *"Since the scope includes [X], models like this usually also
need [Y] — should we add it?"*
