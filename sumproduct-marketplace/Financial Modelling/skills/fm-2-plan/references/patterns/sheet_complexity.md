# Sheet Complexity Guide

When to add sheets beyond the standard 13-14:

| Trigger | Add Sheet | Connect To |
|---------|-----------|-----------|
| 3+ revenue streams | Revenue Assumptions | -> Calculations (revenue section) |
| 2+ debt facilities with covenants | Debt Schedule | -> Calculations (debt section) |
| Construction phase > 12 months | Construction Schedule | -> Calculations (CapEx) |
| Multiple entities | Per-entity Calc sheets + Consolidation | -> Group IS/BS/CFS |
| Lender audience | Cash Waterfall + Covenant Summary | -> CFS, Debt Schedule |
| 3+ scenarios | Scenario Manager | -> General Assumptions (switches) |
| Dashboard required | Dashboard | -> IS, BS, CFS, Key metrics |
| Complex tax (multi-jurisdiction) | Tax Computation | -> Calculations (tax section) |
| Depreciation (5+ asset classes) | Depreciation Schedule | -> Calculations (dep section) |
| Budget model (12+ departments) | Per-department templates | -> Consolidation sheet |
| Commission model (40+ reps) | Rep Data / Territory sheets | -> Commission Calculations |
| Cost allocation | Allocation sheet per method | -> Department P&Ls |
