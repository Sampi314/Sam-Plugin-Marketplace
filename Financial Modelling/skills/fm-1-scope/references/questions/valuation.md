# Valuation (DCF / Comparables) Questions

- What valuation approach? (DCF, comparable companies, comparable transactions, asset-based?)
- Terminal value method? (Gordon Growth, exit multiple?)
- Discount rate — WACC, cost of equity, or specified?
- Valuation date?
- Control premium or minority discount?
- Any non-operating assets to add? (excess cash, investments, property?)
- How should debt be treated? (enterprise to equity bridge)
