## // About

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+what+this+is;%3E+why+it+exists" alt="About typing" />
</p>

A marketplace for Excel and Office plugins where Sumproduct teams and customers discover, install, and update tools centrally - replacing the current state of sharing .xlam files over chat. Primary day-job project at SumProduct.

<a id="features"></a>

## // What's inside

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+features+%C2%B7+goals" alt="Features typing" />
</p>

- `▸` Plugin registry with versioned manifests
- `▸` Distribution API - sign, host, immutable releases
- `▸` Client installer - one-click pull into Excel, auto-update on launch
- `▸` Admin / publisher UI - upload, version, deprecate, rollback
